# 🪮 FINDCELO - 🏝️ Treasure Island

**Treasure Island Game on Celo Blockchain**

A Farcaster Mini App where 6 players choose 1 of 6 lands. 
One land hides the Celo treasure. Winner takes 5x their bet.

## Game Rules

- 6 lands (1-6)
- 3 table types: Bronze (1 CELO), Silver (5 CELO), Gold (10 CELO)
- All 6 lands must be filled
- Smart contract randomly selects treasure land
- Winner gets 5/6 of total pot
- House gets 1/6 commission

## Features

- XP system (1/5/10 XP per game)
- On-chain leaderboard
- Referral system
- Farcaster Frame integration

## Tech Stack

- Celo Blockchain
- Solidity + Hardhat
- Next.js + Frog (Farcaster Frames)
- Neynar API
